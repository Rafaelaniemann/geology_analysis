# geology_analysis

Repositório para análises de descontinuidades geológicas (esqueleto inicial).

Status
- Branch criada: `discontinuities_geology_analisys`.
- Este repositório ainda não contém código de análise completo — este README e os arquivos abaixo são um ponto de partida.

Instalação (exemplo com pip)
1. Crie e ative um ambiente virtual:
   - python -m venv .venv
   - source .venv/bin/activate  (Linux/Mac) ou .venv\Scripts\activate (Windows)
2. Instale dependências:
   - pip install -r requirements.txt

Estrutura sugerida
- README.md

- LICENSE

  - main.py
